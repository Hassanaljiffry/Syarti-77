
# تطبيق سيارتي

تطبيق ذكي لإدارة وصيانة السيارة (الإصدار النهائي)، يشمل:
- اختيار معدل القيادة اليومية (30، 50، 100 كم أو إدخال يدوي).
- حساب مواعيد التذكير.
- بحث عن أكواد الأعطال وعرض النتائج.
- سجل تغييرات قطع الغيار وتوليد تقرير PDF.
- إشعارات للمستخدم.
- تصميم Dark Mode جذاب.
